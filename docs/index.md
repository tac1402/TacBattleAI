# TacLibrary

Статья на хабре описывает мотивацию создания этой библиотеки [TacLibrary открытая библиотека для разработки игр на Unity 3d](https://habr.com/ru/articles/983524/).

В библиотеке выделены три пакета: 

1. TacLibrary.unitypackage - это преимущественно исходный код, полностью все компоненты, которые описаны в данной документации.

2. Rotark.unitypackage - сцена города Ротарк, в которой демонстрируются в основном задачи с симуляцией города, его жителей (NPC), и то что можно описать как экономическое взаимодействие.

3. BattleAI.unitypackage - сцена небольшого поселения, в которой демострируются в основном задачи возникающие при симуляции сражений NPC.

## Какое сейчас состояние библиотеки

Библиотека находится в стадии начальной разработки и ищет своих разработчиков. 

Сейчас большие внимание уделено сцене города Ротарк и задачи перемещения агентов между строениями. 

Сцена BattleAI пока используется меньше, но в ней отрабатывается подход размещения предметов игроком в сцену (добавление укрытий для NPC) с возможностью разместить внутри здания учитывая пересечения со стенами.
