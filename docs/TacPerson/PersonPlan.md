Представляет собой "План персонажа", то какие точки [AgentPoint](../../TacAgent/AgentPoint) и в какой последовательности он должен посетить.

## Конструктор
_PersonPlan(Person argPerson)_

Обязательный параметр ссылка на конкретного персонажа, котором этот план пренадлежит.

## Очередь DayPlan
_Queue<AgentPoint> DayPlan_

Точки агента в порядке необходимости посетить в течении игрового дня.

## Add()
_Add(AgentPoint argAgentPlan)_

Добавить в очередь следующую точку для посещения.

## Remove() 
_AgentPoint Remove()_

Выбрать из очереди посещенную точку.
