---
title: PersonName
---

Функциональность класса PersonName (обеспечивающего создание различных имен персонажам) функционирует без связи с Unity, см. [PersonNameLogic](../../TacPerson/PersonNameLogic). Единственная зависимость возникает при необходимости в Юнити закачать списки имен, которые ожидается найти в текстовых ассетах с путем "Resources\\Data\\Person\\Men" (для мужских имен) и "Resources\\Data\\Person\\Women" (для женских имен). 

Для этого переопределяется метод LoadName(System.Random argRnd), которому нужно передать объект генерирующий случайность.
