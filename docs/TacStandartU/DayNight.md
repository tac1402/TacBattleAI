## gameDays
_Text gameDays_

Текстовое поле в UI в котором будет отображаться текущий номер суток. Нужно задать в редакторе юнити.

## gameTime
_Text gameTime_

Текстовое поле в UI в котором будет отображаться текущие время. Нужно задать в редакторе юнити.

## PauseCompleteStop
_static bool PauseCompleteStop_

Пауза полной остановки. Устанавливается системой сохранений/загрузок. Другие процессы в игре, такие как бесконечные короутины и другие фоновые задачи, должны проверять это значение и останавливать свою работу, чтобы предотвротить обращение к объектам, которые система сохранений может изменить.
 
## Pause
_bool Pause_

Игровая активная пауза, при которое время замирает, но игрок может изучать мир в _застывшем состоянии_.

## PausePress()

Програмный способ вызвать _игровую паузу_ (Pause = true) или снять с паузы (Pause = false), если пауза уже была установлена. Используется компонентом самостоятельно при нажатии кнопкина клавиатуре __Pause__.
